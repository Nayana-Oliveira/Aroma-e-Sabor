# Aroma & Sabor (Projeto Acadêmico - Programação Web)

Projeto acadêmico "Aroma & Sabor" para a matéria de Programação Web. Foca em aplicar conceitos de HTML semântico, CSS (Flexbox/Grid) e manipulação de DOM com JavaScript básico (eventos e alertas), sem `localStorage` ou backend.

## Objetivo do Projeto

Este projeto foi desenvolvido como atividade para a disciplina de **Programação Web**. O objetivo principal é aplicar na prática os conceitos fundamentais de desenvolvimento front-end aprendidos em sala de aula.

O foco não está em criar um e-commerce funcional com persistência de dados (como `localStorage` ou banco de dados), mas sim em demonstrar competência nas seguintes áreas:

1.  **Estruturação HTML:** Uso correto de tags semânticas (`<header>`, `<main>`, `<footer>`, `<section>`, `<nav>`).
2.  **Estilização CSS:** Criação de um layout moderno e responsivo, utilizando **Flexbox** e **Grid**, variáveis CSS (`:root`), importação de fontes externas e estilização de componentes.
3.  **JavaScript Básico:** Manipulação de eventos (`onclick`, `DOMContentLoaded`) e feedback ao usuário (manipulação de DOM e `alert()`).

## Funcionalidades Aplicadas

* **Navegação Multi-página:** Estrutura de 3 páginas (`index.html`, `produtos.html`, `pedido.html`) conectadas por um menu de navegação.
* **Design Responsivo:** O layout se adapta a desktops e dispositivos móveis usando Media Queries.
* **Landing Page Atraente:** A `index.html` serve como uma vitrine de boas-vindas.
* **Simulação de Interatividade:** Os botões "Adicionar" e "Finalizar" disparam alertas de feedback, demonstrando a captura de eventos de clique.
* **Página de Pedido Estática:** A `pedido.html` exibe uma mensagem de "carrinho vazio", pois o escopo do projeto não inclui a persistência de dados entre as páginas.

## Como Executar

Este é um projeto 100% estático (somente front-end).

1.  Faça o download ou clone este repositório.
2.  Abra a pasta do projeto.
3.  Dê um duplo clique no arquivo `index.html` para abri-lo no seu navegador.
